# Netflix Data Analysis Project 🎬

## Overview
In this project I analyzed the Netflix titles dataset using Python, Pandas, NumPy, 
Matplotlib and SQL. The dataset contains 8807 Netflix titles including Movies and 
TV Shows from various countries around the world.

## Questions Answered
1. What is the ratio of Movies vs TV Shows on Netflix?
2. Which countries produce the most content on Netflix?
3. What is the most common age rating on Netflix?
4. How much content was added each year?
5. Which directors have the most titles on Netflix?

## Key Findings
- Netflix has 70% Movies and 30% TV Shows
- USA dominates with 2809 titles followed by India with 972 titles
- Over 60% of Netflix content targets mature audiences (TV-MA and TV-14)
- 2019 was Netflix peak year with 2016 titles added
- COVID-19 impacted content production in 2020 and 2021
- Rajiv Chilaka tops directors list with 19 titles — mostly animated content

## Tools Used
- Python 3
- Pandas
- NumPy
- Matplotlib
- SQL (SQLite)
